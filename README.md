# EDC CBIT 🚀

### Premium Startup-Style Website for Entrepreneurship Development Cell, CBIT

A modern, visually striking website concept built for the **Entrepreneurship Development Cell (EDC), Chaitanya Bharathi Institute of Technology**.

This project reimagines how college clubs and student communities can present themselves online — with a sleek startup-grade brand presence instead of a basic traditional website.

---

## ✨ Key Features

* ⚡ Bold premium hero section
* 🎯 Smooth animations & hover interactions
* 👥 Team showcase layout
* 🌙 Modern dark aesthetic UI
* 📱 Responsive design
* 🚀 Startup-inspired visual identity
* 🧩 Built with scalable React architecture

---

## 🚀 Run Locally

```bash
git clone https://github.com/KjvRajesh11/EDC_CBIT.git
cd EDC_CBIT
npm install
npm run dev
```

Then open:

```text
http://localhost:5173
```
